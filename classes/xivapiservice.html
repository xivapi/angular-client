searchCharacter

  • searchCharacter(name: string, server?: undefined | string, page?: undefined | number): Observable<CharacterSearchResult>
    • Defined in xivapi.service.ts:160

    Search for a character on The Lodestone. This does not search XIVAPI but instead it goes directly to lodestone so the response will be "real-time". Responses are cached for 1 hour, it is important to know that Lodestone has a ~6 hour varnish and CDN cache.

    Parameters

    • name: string

      The name of the character to search, you can use + for spaces or let the API handle it for you. If you search very short names you will get lots of responses. This is an issue with The Lodestone and not much XIVAPI can do about it at this time.

    • Optional server: undefined | string

      (optional) The server to search against, this is case sensitive. You can obtain a list of valid servers via getServerList method.

    • Optional page: undefined | number

      Search or move to a specific page.

    Returns Observable<CharacterSearchResult>

searchFreeCompany

  • searchFreeCompany(name: string, server?: undefined | string, page?: undefined | number): Observable<object>
    • Defined in xivapi.service.ts:184

    Search for free companies on The Lodestone. This parses the lodestone in real time so it will be slow for non-cached responses. All search queries are cached for 1 hour, it is important to know that Lodestone has a ~6 hour varnish and CDN cache. This does not search XIVAPI so free companies found may not be on the service and will be added when requested by their specified lodestone ID.

    Parameters

    • name: string

      The name of the company to search, you can use + for spaces or let the API handle it for you. If you search very short names you will get lots of responses. This is an issue with The Lodestone and not much XIVAPI can do about it at this time.

    • Optional server: undefined | string

      (optional) The server to search against, this is case sensitive. You can obtain a list of valid servers via getServerList method.

    • Optional page: undefined | number

      Search or move to a specific page.

    Returns Observable<object>
